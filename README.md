tumblr_theme
============

#STATUS: ongoing

This is a very simple Tumblr theme. Feel free to use it in your own projects.

NOTE!
This theme uses CSS3 Flexbox -model and it is supported only in latest browsers.

 - Chrome 21+
 - Safari 6.1+
 - Firefox 22+
 - Opera 12.1+
 - IE 11+
 - Android 2.1+
 - iOS 7.1+
 - WindowsPhone 11+
